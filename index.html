Schedule:

Week 1 | Overview

We'll do intros and run a quick, in-class data experiment. Then, I'll present some examples of compelling data and visual storytelling.

Order of operations:

  • Introductions
  • Syllabus review
  • Intro to data and visual journalism
  • In-class assignment: graphics critique

Readings for week 2:

  • TK
Week 2 | Speed data-ing

We'll find some data, then tease out the really interesting bits using basic spreadsheet functions. You'll learn how to do some basic math that will make your life with data a whole lot easier.

Order of operations:

  • What is data and where can we find it?
  • Basic Excel tutorial
  • Basic numeracy
  • In-class assignment: What's the story in the data?

Readings for week 3:

  • TK
Week 3 | Charting data (design principles for web and mobile)

We'll go over different types of charts and the fundamentals of good data viz design.

Order of operations:

  • What is data and where can we find it?
  • Basic Excel tutorial
  • Basic numeracy
  • In-class assignment: What's the story in the data?
Week 4 | Mapping data 1 (design principles for web and mobile)

We'll cover the fundamentals of mapping.

Order of operations:

  • When to map data – and when not to
  • Organizing information spatially
  • Design: grouping, hierarchy, color
  • In-class assignment: Data mapping with Fusion Tables???

Prep for week 5:

  • Install Sublime Text (trial)
  • Sign up for Github (free)
  • Sign up for jsfiddle (free)
Week 5 | Intro to HTML/CSS 1

You'll learn how to make a basic HTML web page and then class it up with CSS (ahem, you'll appreciate the pun later).

Order of operations:

  • Learn to say, "Hello world!"
  • Styling with CSS (inline and external)
  • 10(ish) CSS selectors you need to know now
  • Uploading your site to the web
Week 6 | Intro to HTML/CSS 2

Building on last week's class, you'll learn how to build better web pages.

Week 7 | Mapping data 2
